## 🌟 Key Features Explained

### 🔥 Streak System
- Track consecutive days with completed tasks
- Build motivation through consistency
- Earn bonus XP at 7-day milestones
- See your "Best Streak" achievement

### ⭐ XP & Leveling
```
Complete Task       →  +10 XP
Focus Session       →  +25 XP
Goal Step           →  +5 XP
7-Day Streak        →  +50 XP (Bonus!)

Level = √(XP/10) + 1

Level  1: 0-90 XP
Level  5: 250+ XP
Level 10: 1000+ XP
```

### 🎯 Goal Breakdown (The Secret Weapon!)
**Problem:** "Learn Spanish" feels overwhelming

**FocusFlow Solution:**
1. Create goal: "Learn Spanish"
2. Break into steps:
   - Learn 100 words
   - Complete Duolingo course
   - Practice 10 conversations
   - Watch Spanish show
3. Push today's step → Tasks: "Learn 10 words"
4. Complete small step daily
5. Big goal achieved through tiny actions!

**Why it works:** Big goals → Small steps → Daily tasks → Actual progress

### ⏱️ Pomodoro Timer
- Select a task from your list

## 💡 Best Practices for Maximum Results

### The 3-5 Rule
**Don't overwhelm yourself!**
- Morning: Add 3-5 tasks maximum
- Prioritize ruthlessly
- Focus on impact, not quantity

### The Goal Staircase
```
Level 1: Big Goal (e.g., "Get fit")
Level 2: Monthly Milestones (e.g., "Workout 12x this month")
Level 3: Weekly Steps (e.g., "Gym 3x this week")
Level 4: Daily Actions (e.g., "30 min workout today")

Start at Level 4, climb to Level 1!
```

### The Evening Ritual
**Every day at 6 PM:**
1. Review what you completed (celebrate!)
2. Clear completed tasks
3. Plan tomorrow (3-5 tasks)
4. Add one goal step for tomorrow
5. Close laptop feeling accomplished

### The Focus Formula
```
Morning Energy (8-11 AM):
→ 2-3 Pomodoro sessions
→ Hardest tasks first
→ Deep work mode

Afternoon (2-5 PM):
→ 1-2 Pomodoro sessions
→ Medium priority tasks
→ Collaborative work

Evening (6-8 PM):
→ Light tasks
→ Planning
→ Learning/growth activities
```

### The Streak Strategy
**To build unbreakable streaks:**
1. Make it easy: Set ONE must-do task
2. Make it visible: Open FocusFlow first thing
3. Make it rewarding: Celebrate every check-off
4. Make it automatic: Add to startup (optional)

**Emergency Streak Saver:**
Even on tough days, complete 1 tiny task:
- "Review tomorrow's plan" (counts!)
- "Organize one folder"
- "Read one article"

Streak maintained = Momentum preserved!

## 🔧 Troubleshooting Quick Reference

### Notifications Not Working
```
Browser:
→ Settings → Notifications → Allow FocusFlow

Windows:
→ Settings → Notifications → Enable for PowerShell
```

### Data Not Saving
```
→ Don't use Incognito mode
→ Open same FocusFlow.html file each time
→ Check browser storage settings
```

### Timer Won't Start
```
→ Refresh the page
→ Select a task first
→ Check browser console (F12)
```

### Hourly Reminders Not Appearing
```
→ Run: Get-ScheduledTask -TaskName "FocusFlow Hourly Reminder"
→ Check Windows Notifications enabled
→ Re-run Setup-FocusFlow-Reminders.ps1
```
