## Feature Illustrations

### 🔥 Streak System

```
Day 1:  ☑ Task completed → Streak: 1 🔥
Day 2:  ☑ Task completed → Streak: 2 🔥🔥
Day 3:  ☑ Task completed → Streak: 3 🔥🔥🔥
```

### ⭐ XP Progression

```
Level 1 ─────────────> Level 2 ─────────────> Level 3
   0 XP              90 XP              190 XP

Actions:
┌─────────────────────────────────────────────┐

More...  230 XP (Level 6)
```

## Interaction Patterns

### Task Completion Animation

```
1. User clicks checkbox
         │
         ▼
2. Checkbox animates ☐ → ☑
         │
         ▼
3. Confetti launches from center
         │
         ▼
4. Success sound plays
         │
         ▼
5. XP counter increments (+10)
         │
         ▼
6. Task fades to 60% opacity
         │
         ▼
7. Text gets strikethrough
         │
         ▼
8. Random motivational quote changes
```

### Focus Timer States

```
STATE 1: READY
┌───────────────┐
│    25:00      │  ← Big display
│   [Start]     │  ← Green button
└───────────────┘

STATE 2: RUNNING
┌───────────────┐
│    18:42      │  ← Counting down
│ [Pause] [↻]   │  ← Pause & Reset
└───────────────┘

STATE 3: PAUSED
┌───────────────┐
│    18:42      │  ← Frozen time
│ [Start] [↻]   │  ← Resume or Reset
└───────────────┘

STATE 4: BREAK
┌───────────────┐
│  BREAK! 5:00  │  ← Break message
│   [Start]     │  ← Start next session
└───────────────┘
```
